# 前端是代码届的导演

- html 为演员， css 为化妆师， js 为动作导演

- css 选择器
    1. 标签选择器
    2. 类名选择器 class .  一类元素 []
    3. id选择器 #  一个元素  唯一的
    4. 属性选择器  []
    5. >   子元素
    6. 空格选择器   div  p  内部 无论多少级别  DOM树状结构
    7. 选择  兄弟节点
    8. ~  *  ：  .....
    可以使用多个规则嵌套
        1. 选择越来越准确
        2. 优先级  权重  计算公式
            标签  1分
            类名  10分
            id    100分
            行内样式  ...
            ！important  无限大

- emmet 使用 css 选择器作为快捷输入的规则
        1. 选择器： .anony-nav-links>ul>(li>a)*9
        2. 属性选择器：  .anony-nav-links>ul>(li>a[herf][class][target])*9
